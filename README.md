**Tumbang Preso: Street King Arcade** is a high-fidelity, Filipino-themed 2D arcade game built with native PHP and Vanilla JavaScript. Experience the thrill of the streets as you aim, throw, and outsmart the "Taya" in this digitized version of the classic Filipino childhood game.

---

## 🎮 Game Features

*   **Arcade-Perfect Aiming**: A precision-based gauge system with 'Snap-to-Target' logic for perfect Cyan timing.
*   **Performance Ranking**: Get rated from **Rank F** to **Rank S (Street King)** based on your health and score.
*   **Dynamic AI (Taya)**: Face off against an AI bot that chases you, recovers the can, and stuns upon being hit.
*   **High-Fidelity Assets**: Custom top-down background art and procedural animations for a smooth street-vibe experience.
*   **Banging Soundtrack**: Upbeat arcade background music and satisfying SFX for every throw and hit.

| Action | Key / Control |
| :--- | :--- |
| **Move** | `W`, `A`, `S`, `D` |
| **Jump** | `Space` |
| **Aim/Throw** | `Click & Drag` (Slipper) |
| **Release** | `Mouse Up` (Check the Gauge!) |

---

## 🎯 How to Score

1.  **Throw** your tsinelas to knock down the **Lata** (Can).
2.  Once down, **run** to pick up your tsinelas.
3.  **Return** to the safe zone (Right Side) while the lata is down to score a point.
4.  **Avoid being tagged** by the Taya while you are in the court!

---

## 🛠️ Technical Stack

*   **Frontend**: HTML5 Canvas, Vanilla JavaScript (ES6+), CSS3.
*   **Backend**: Native PHP 8.x.
*   **Database**: MySQL (Leaderboard & Stats).
*   **Audio**: HTML5 Audio API (Stabilized Hosting).

---

## 🚀 Installation (Local)

1.  Clone the repository to your `htdocs` folder.
2.  Import `db_game.sql` into your MySQL server (via phpMyAdmin).
3.  Configure `core/db.php` with your database credentials.
4.  Open `http://localhost/Tumbang-Preso/` in your browser.
